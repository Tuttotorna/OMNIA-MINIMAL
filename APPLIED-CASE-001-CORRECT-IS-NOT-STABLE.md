# Applied Case #001 — Correct Is Not Stable

## Purpose

This case is the minimal public demonstration of OMNIA-MINIMAL after the Infinity-First reframe.

The goal is to show one thing clearly:

A locally correct answer can become structurally unstable when the field expands.

This is the smallest practical expression of the Law of Totality.

## The local answer

A system gives an answer.

The answer is correct under the local condition.

The local test passes.

The local metric is satisfied.

The local observer says:

The system works.

This may be true.

But it is not enough.

## The missing distinction

Correctness and stability are different properties.

Correctness means:

the output matches a local condition.

Stability means:

the output remains coherent when the field expands.

A correct answer can be unstable.

A useful answer can be unstable.

A plausible answer can be unstable.

A statistically true number can be unstable.

A working process can be unstable.

A model can be correct in one frame and collapse in another.

## Minimal example

Suppose an answer is correct for one exact prompt.

Now ask the same question with equivalent wording.

If the answer changes in meaning, direction, or structural conclusion, the first answer was not stable.

It may have been locally correct.

It was not structurally stable.

The failure is not necessarily in the content.

The failure is in treating one local output as if it were the whole object.

## Mathematical-statistical example

In Simpson's Paradox, an aggregate result can be arithmetically true and decisionally wrong.

The aggregate can say:

Treatment B wins.

But after restoring the missing structure, each relevant subgroup can show:

Treatment A wins.

The aggregate was not false as arithmetic.

It was false as a total decision object.

This is the minimal OMNIA distinction.

A local number can be true.

A decision based on treating it as total can be wrong.

## AI example

An AI answer can be correct in one response.

But if it collapses the user's original frame into a familiar category, it becomes structurally false.

The words may remain.

The field is gone.

The answer is locally fluent.

It is not faithful to the originating structure.

Again:

correct is not stable.

## Structural diagnosis

The error appears when this local form:

the answer passed the immediate test

is treated as this total claim:

the answer is structurally valid.

That substitution is the false totalization.

OMNIA-MINIMAL exists to block that substitution.

## Minimal diagnostic rule

Do not ask only:

Is it correct?

Ask:

What field was excluded in order for it to look correct?

Then ask:

Does it remain coherent when that field is restored?

If it breaks, it was locally correct but structurally unstable.

## Core sentence

Correct is not stable.

## Expanded sentence

A finite correctness becomes structurally false when it is treated as total.

## Final sentence

OMNIA-MINIMAL is the smallest operational proof that local correctness and structural stability are not the same thing.
