# Extended Results

The v0.4 dataset extends the minimal demonstration beyond the original three cases.

The aim is still minimality.

The repository now includes several transformation types:

    arithmetic variation
    symbolic reordering
    paraphrase
    distractor clause
    representation change

---

## Expected signal distribution

The expected output contains:

    STABLE
    FRAGILE
    DIVERGENT

The important point is that FRAGILE appears in more than one transformation family.

This supports the narrow claim:

    surface correctness can pass while structural stability fails

---

## What this strengthens

v0.1 showed the basic distinction.

v0.2 added continuous validation.

v0.3 added paper-style framing.

v0.4 strengthens the dataset while keeping it small.

---

## What this does not prove

This does not prove universal OMNIA validity.

This does not prove that every fragile signal is semantically important.

This does not prove that a stable signal is true.

It only strengthens the minimal operational claim:

    correctness is not structural stability
