## Boundary

OMNIA-MINIMAL points to the wider ecosystem.

It does not replace it.

The correct reading order is:

    first understand the minimal claim
    then inspect the demo
    then inspect the case catalog
    then move outward to the full modules

The claim remains:

    correctness is not structural stability

The boundary remains:

    measurement != inference != decision
